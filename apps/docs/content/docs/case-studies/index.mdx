Every case uses both kinds of tool:

- agent-browser or agent-device performs the action and checks visible UI.
- Genie reads the live React and TanStack evidence.

The commands and outputs below came from live demo runs. Component IDs and timings change between
sessions, so always discover the current ID first.

These live cases cover the Vite, TanStack Start, Router, plain web, and Expo demos. The Next.js page
is a setup guide and does not claim a live case.
